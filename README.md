# Instruction setup mOS 18.04 X86 MDEC
```
apt install git -y && \
git clone https://github.com/mscits/mOS-18.04-X86-MDEC.git /tmp/mOS-18.04-X86/ && \
chmod +x /tmp/mOS-18.04-X86/setup.sh && \
sh /tmp/mOS-18.04-X86/setup.sh
```
